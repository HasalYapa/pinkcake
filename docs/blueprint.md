# **App Name**: CakesLK

## Core Features:

- Cake Showcase: Display a modern landing page featuring a hero section with showcased cakes, categorized into Birthday, Bento, Wedding, and Custom cakes.
- Order Submission: Enable customers to select cake category/size/flavor, add a message, pick a delivery date/location, and upload a reference image. Auto price calculation and generation of a unique Order ID upon submission.
- WhatsApp Order Confirmation: Generate a WhatsApp link to confirm the order.
- Order Tracking: Allow users to track their order status in real-time via an Order ID.
- Secure Admin Authentication: Implement secure authentication for admins using Supabase Auth, protecting admin-only routes.
- Order Management Dashboard: Provide an admin dashboard to view/update order status (Pending, Accepted, Baking, Ready, Delivered) and payment status (Pending, Paid), filter by date, and delete orders.
- AI Cake Suggestion Tool: Suggest cakes to the user using AI depending on the user's selections for a more tailored user experience.

## Style Guidelines:

- Primary color: Warm terracotta (#E07A5F), evoking Sri Lanka's rich earth tones.
- Background color: Light beige (#F8F2EC), a desaturated version of terracotta to give warmth and allow the cake images to stand out.
- Accent color: Muted olive green (#81B29A), an analogous color that adds a touch of freshness.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look with warmth and personality.
- Use simple, clean icons from Tailwind UI, customized to match the overall aesthetic.
- Mobile-first, clean layout leveraging Tailwind CSS grid and flexbox for responsiveness.
- Subtle transitions and animations using Tailwind CSS transitions and transforms for a polished user experience.
